Texto marcado

Podemos criar também um texto marcadousando a tag MARK

Podemos alterar a cor a marcação porem isso já é CSS

Como por exemplo um texto com cor diferente

Para colocar letras pequenas

Nesse caso um termo pequeno no texto é usado a tag SMALL

Texto deletado

Podemos marcar um texto como excluido para indicar que ele deve ser lido, mais não considerado.

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado

Texto sobrescrito

Para inserir coisas do tipo x20+3 no caso sendo x elevado a 20 +3, basta por a tag SUP ou envelopar a letra ou numero desejado

Texto subscrito

Para inserir coisas do tipo H2O , basta por a tag SUB ou envelopar a letra ou numero desejado
